Detailed Description

Asynchroniczne pobieranie danych z API GIOŚ — nie blokuje GUI.

WIELOWĄTKOWOŚĆ: QtConcurrent::run() uruchamia każde żądanie HTTP w osobnym wątku z puli wątków Qt. GUI wątek nigdy nie czeka na sieć. QFutureWatcher emituje sygnały Qt gdy wątek kończy pracę (wzorzec Observer).

Obsługa offline: przy błędzie sieci automatycznie próbuje wczytać ostatnie zapisane dane z lokalnego cache JSON.
