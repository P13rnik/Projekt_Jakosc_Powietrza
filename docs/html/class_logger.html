Detailed Description

Klasa narzędziowa do logowania zdarzeń aplikacji.

Loguje komunikaty z poziomem ważności (INFO, WARNING, ERROR) do konsoli debugowej Qt. Możliwe rozszerzenie o zapis do pliku.
