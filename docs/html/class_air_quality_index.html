Detailed Description

Model indeksu jakości powietrza dla stacji pomiarowej.

Odpowiada odpowiedzi endpointu /index/getIndex/{stationId} z API GIOŚ. Zawiera ogólny indeks dla stacji oraz indeksy cząstkowe per parametr.
