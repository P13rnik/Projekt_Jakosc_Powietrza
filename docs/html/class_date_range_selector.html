Detailed Description

Widget umożliwiający wybór zakresu czasowego danych na wykresie.

Udostępnia predefiniowane okresy (ostatnie 24h, 48h, 7 dni) oraz możliwość ręcznego ustawienia daty „od" i „do". Po zmianie zakresu emituje sygnał rangeChanged().
