Detailed Description

Widget wyboru czujnika (parametru) z listy stanowisk stacji.

Wyświetla listę dostępnych parametrów pomiarowych (PM10, PM2.5, NO2, O3, SO2, C6H6) i emituje sygnał sensorSelected() gdy użytkownik zmieni wybór.
